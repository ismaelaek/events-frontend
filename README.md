# Events Managment - Frontend

## installation 

```shell
git clone https://github.com/ismaelaek/events-frontend.git

cd events-frontend
```
if you are not using Laravel Herd, make sure to change `baseURL` in `src\api\client.js`.

### install reqs
```shell
npm install 
```

### run app

```shell
npm run dev
```
